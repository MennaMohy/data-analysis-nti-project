# Customer Data Analysis Project

## 📌 Project Overview
This project focuses on analyzing a synthetic customer dataset to understand customer behavior, purchasing patterns, and product preferences. The goal is to clean the data, explore key trends, and extract meaningful insights using Python.

## 🛠️ Tools & Technologies
- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn
- Google Colab

## 📊 Project Steps

### 1. Data Cleaning
- Removed irrelevant columns
- Handled missing values in Age, Gender, PurchaseAmount, Rating, etc.
- Standardized inconsistent gender values
- Corrected invalid age values
- Converted data types and cleaned date column

### 2. Exploratory Data Analysis (EDA)
- Analyzed age distribution of customers
- Studied gender distribution and purchase behavior
- Identified most purchased product categories
- Explored relationship between age and purchase amount
- Analyzed yearly purchase trends
- Investigated rating distribution and category performance
- Identified top 5 customers based on total purchases

### 3. Data Visualization
Created multiple visualizations using Matplotlib and Seaborn including:
- Histograms
- Bar plots
- Scatter plots
- Line charts
- Box plots

## 📈 Key Insights
- Certain age groups contribute more to overall purchases
- Clear differences in spending behavior between genders
- Some product categories significantly outperform others
- Top customers contribute a large portion of total revenue
